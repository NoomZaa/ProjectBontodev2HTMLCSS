Project Book Center

คู่มือการใช้งานโปรแกรม Book Center

1.หน้าต่าง Login ให้ผู้ใช้ดำเนินการกรอก User และ Password ของตนเองเพื่อเข้าสู่ระบบ

2.หน้าต่างของ Admin เมื่อผู้ใช้งานดำเนินการกรอก User และ Password เสร็จแล้วถ้าตรวจสอบพบว่าสถานะคือ Admin ระบบจะดำเนินการเปิดหน้า User Admin ให้

3.หน้า Admin Customer ในหน้าต่างนี้ผู้ดูแลระบบสามารถค้นหาชื่อของ User หรือจะเป็นการ เพิ่ม User,แก้ไข User และลบ User ได้

4.หน้า Admin Book ผู้ดูแลระบบสามารถค้นหาหนังสือ หรือ เพิ่มหนังสือ,แก้ไขหนังสือ และลบหนังสือได้

5.หน้าต่างของ User ในกรณีที่ดำเนินการ Login แล้วระบบตรวจสอบสถานะพบเป็น User ระบบจะส่งผู้ใช้งานมาหน้านี้ ซึ่งสามารถเลือกได้เพียงหัวข้อเดียวคือสั่งซื้อหนังสือ

6.หน้า User Buy หน้าที่ผู้ใช้งานไว้ใช้เลือกซื้อหนังสือที่ต้องการ โดยระบบสามารถค้นหาหนังสือได้ด้วยรหัสหนังสือ,ชื่อหนังสือ,รายละเอียดหนังสือ และราคา โดยเมื่อผู้ใช้งานดำเนินการเลือกหนังสือแล้วให้กรอกจำนวนหนังสือ ระบบจะทำการคำนวณราคาให้ เมื่อได้หนังสื่อแล้วให้ดำเนินการกดสั่งซื้อ เป็นอันเสร็จขั้นตอน

7.หน้าต่างแสดง MessageBox สรุปรายการสั่งซื้อ
